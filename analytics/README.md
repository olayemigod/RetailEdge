# Analytics Starter Kit

This folder holds reusable context for future data work.

## What to keep here

- `data_context.md`: durable business and data context
- `metric_catalog.md`: agreed metric definitions
- `analysis_brief_template.md`: template for new analysis requests

## How to use this

1. Fill in `data_context.md` with your systems, tables, owners, and caveats.
2. Define important KPIs in `metric_catalog.md` before doing repeated analysis.
3. Copy `analysis_brief_template.md` when you want help with a new question.

## Why this helps

Reusable context makes future data work faster and more reliable because we can:

- use the same source-of-truth tables each time
- avoid redefining metrics on every request
- capture caveats once and reuse them
- compare future analyses against a stable business context
